# Workflow

## Step-1:
### Pull the latest code before doing anything!
> git **checkout** main
>
> git pull **origin** main

## Step-2:
### If you want to work on any thing or want to mark it for others, create an issue. Then either claim it yourself or someone else will claim it. This prevents multiple people from working on the same topic.
>* Go to issues
>* Create an issue
>* Claim it or leave it for others

## Step-3:
### Create a temp branch for the issue that you are working on. Do the changes, then create a pull request from your temp branch to the main branch. **Never commit to the main branch directly**.
Steps to create a temp branch
>* git checkout -b \<temp-branch-name\>
